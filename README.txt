Descripción del Proyecto
Este es un sistema de gestión de un hotel de 5 estrellas que permite a los usuarios gestionar reservas de habitaciones y registrar nuevos usuarios. El proyecto tiene como objetivo proporcionar una interfaz fácil de usar para la administración de clientes, la gestión de reservas y la creación de usuarios, lo que garantiza una experiencia eficiente y optimizada tanto para los empleados del hotel como para los huéspedes.

Funcionalidades Principales
Gestión de Usuarios:

Registro de nuevos usuarios (clientes y empleados).
Inicio de sesión seguro.
Roles diferenciados (usuarios, administradores).
Gestión de Habitaciones:

Visualización de habitaciones disponibles.
Reserva de habitaciones según tipo y disponibilidad.
Cancelación de reservas.
Gestión de Reservas:

Creación, modificación y cancelación de reservas.
Verificación de disponibilidad en tiempo real.
Envío de confirmación de reserva por correo electrónico (pendiente).
